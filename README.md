O Festivite é uma plataforma intuitiva para a criação de convites digitais personalizados. Com ele, usuários podem transformar os detalhes de seus eventos em convites visuais elegantes, prontos para compartilhamento, garantindo que a primeira impressão da sua festa seja inesquecível.
✨ Funcionalidades
Customização Completa: Edite título, data, local e descrição do evento.
Design Flexível: Escolha entre diferentes temas, paletas de cores e imagens de fundo.
Informações de Contato: Adicione dados para RSVP e contato direto.
Fluxo de Segurança: Aceite de termos e condições antes da geração do arquivo final.
Geração de Convite: Exportação do convite personalizado para uso imediato.

🛠️ Tecnologias Utilizadas

Frontend: React.js / Vue.js / HTML5 & CSS3
Estilização: Tailwind CSS / Styled Components
Gerenciamento de Estado: Context API / Redux
Ícones: Lucide React / Font Awesome
